Architecture Overview

  • AWS infrastructure provisioned using Terraform
  • Ansible for configuration management
  • Dockerized web application
  • Prometheus & Grafana monitoring
  • Secure access via Cloudflare Tunnel

Monitoring

Prometheus collects metrics from the web server (CPU, RAM, Disk). Grafana visualizes these metrics and is exposed securely using Cloudflare Tunnel.

CI/CD

GitHub Actions is used to automatically deploy this documentation to GitHub Pages whenever changes are pushed to the main branch.
